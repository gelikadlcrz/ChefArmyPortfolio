# Army S. Barnachea — Culinary Portfolio Redesign

A premium, editorial-style culinary portfolio for Army S. Barnachea, built with React, TypeScript, Vite, CSS, and Lucide React.

## What is included

- Fully redesigned single-page portfolio
- High-end dark/champagne visual direction
- Responsive layout for desktop, tablet, and mobile
- Curated food showcase with category filters
- About, career direction, skills, credentials, and contact sections
- Optimized WebP assets from the provided image archive
- Downloadable resume PDF inside `public/assets`

## Tech stack

- React
- TypeScript
- Vite
- CSS
- Lucide React

## Run locally

```bash
npm install
npm run dev
```

## Build for deployment

```bash
npm run build
```

The production files will be generated in the `dist` folder.
